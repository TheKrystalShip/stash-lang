# Stash

A C-style interpreted shell scripting language that combines shell scripting power with structured data capabilities.

## Building

```bash
dotnet build Stash/
```

## Running

```bash
# Start the REPL
dotnet run --project Stash/

# Or after building
./Stash/bin/Debug/net10.0/Stash
```
